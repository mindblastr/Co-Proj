mnist2pgm.c              C program (for PC) that reads a mnist image file 
                         (see http://yann.lecun.com/exdb/mnist/) 
                         and prints image N in pgm format to stdout.
                         (see http://paulbourke.net/dataformats/ppm/)
mnist2pgm.exe            Windows executable
t100-images-idx3-ubyte   Binary file with first 100 images of MNIST set.
